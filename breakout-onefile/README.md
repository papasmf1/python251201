# Breakout Game

This is a simple implementation of the classic Breakout game using Python and Pygame.

## Installation

To run this project, you need to have Python installed on your machine. You can download it from [python.org](https://www.python.org/downloads/).

### Dependencies

This project requires the following Python package:

- `pygame`

You can install the required dependencies using pip. Run the following command in your terminal:

```
pip install -r requirements.txt
```

## Running the Game

To start the game, run the following command in your terminal:

```
python breakout.py
```

## Controls

- Use the left and right arrow keys to move the paddle.
- Press the spacebar to start the game.

## Game Objective

The objective of the game is to break all the bricks by bouncing the ball off the paddle. Try to clear all the bricks without letting the ball fall off the screen!
